       STEP 1  ORCHESTRATOR → HANDOVER   📚 delegate_research(...)
           ├─ RESEARCHER  🔍 web_search(...)
           ├─ OBSERVE     ← population result
           └─ RETURN      ← facts handed back
       STEP 2  ORCHESTRATOR → HANDOVER   🧮 delegate_analysis(...)
           ├─ ANALYST  🧮 calculator(...)
           └─ RETURN      ← expression = result
       STEP 3  ORCHESTRATOR → HANDOVER   ✍️ delegate_writing(...)
           ├─ WRITER  draft v1
           ├─ CRITIC  ✗ revise / ✓ approved
           └─ RETURN      ← brief ready
  3. Green FINAL REPORT panel with facts, calculation, brief, revisions
  4. Footer: "3 handovers · N sub-agent tool calls · N tokens · Xs"

SAY:
  "This is orchestrator–workers plus evaluator–optimiser from slide 6. The
   orchestrator doesn't do the work — it hands off to specialists and gets
   control back. Writer and critic loop until the brief passes review.
   Notice it costs more tokens than the single agent — multi-agent is a cost,
   not a badge of honour."
